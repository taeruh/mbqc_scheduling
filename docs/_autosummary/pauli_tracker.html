pauli_tracker

Wrapper around the essential functionality of the pauli_tracker crate, and the mbqc_scheduling crate.

When exporting Rust code through the FFI, we loose the ability to be generic. Because of that we can only support specific types in this wrapper. The submodule structure kinda emulates these types. For example, the Live in live.map corresponds to Rust’s Live<Map<_>> type.
